# Exercise 106 - Adding Objects

## 📚 Objective

Add properties to objects using JavaScript object literal syntax.

## 🧠 Concepts

- Object literals
- Properties
- Key-value pairs

## 💻 Solution

```javascript
const user = {
  id: 1,
  name: "Alexandre",
};

user.email = "example@fakemail.com";

console.log(user);
```

## ▶️ Output

```text
{ id: 1, name: 'Alexandre', email: 'example@fakemail.com' }
```

## 📖 Explanation

This exercise demonstrates how to add new properties to an existing JavaScript object using dot notation.

## ✅ What I Learned

- How to add new properties to an object.
- How to access object properties using dot notation.
- Why objects are useful for storing related data.
