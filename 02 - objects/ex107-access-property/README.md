# Exercise 106 - Accessing Objects

## 📚 Objective

Access a properties of object in console using JavaScript object literal syntax.

## 🧠 Concepts

- Object literals
- Properties
- Key-value pairs

## 💻 Solution

```javascript
const user = {
  id: 1,
  name: "Alexandre",
};

console.log(user.name);
```

## ▶️ Output

```text
Alexandre
```

## 📖 Explanation

This exercise demonstrates how JavaScript objects store data as key-value pairs.
