# Exercise 106 - Creating Objects

## 📚 Objective

Create an object using JavaScript object literal syntax.

## 🧠 Concepts

- Object literals
- Properties
- Key-value pairs

## 💻 Solution

```javascript
const user = {
  id: 1,
  name: "Alexandre",
};
```

## ▶️ Output

```text
{ id: 1, name: 'Alexandre' }
```

## 📖 Explanation

This exercise demonstrates how JavaScript objects store data as key-value pairs.

- Why objects are useful.
