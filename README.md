# JavaScript Exercises Collection

## 🧩 Introduction

This repository contains my JavaScript exercise collection. Its purpose is to help me practice, improve my programming skills, and strengthen my understanding of JavaScript.

I created this repository to study the fundamental concepts of the language and the tools that are commonly used in real-world projects. Throughout this journey, I use high-quality references and the technical guidance provided by my university.

## 📚 Objective

The main objectives of this repository are to:

- Learn JavaScript through consistent practice.
- Develop problem-solving and programming logic skills.
- Improve my Git and GitHub workflow.
- Create well-documented projects using Markdown (`README.md`).
- Build a portfolio that demonstrates my progress as a Web Developer.

## ▶️ Install and test

1. clone repository:

```bash
git clone https://github.com/YOUR_USERNAME/REPOSITORY_NAME.git
cd REPOSITORY_NAME
```

2. add link in 'src="ex123/solution.js"' from index.html arquive
3. run index.html in your browser.
